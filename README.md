# Alutamarket Analytics Dashboard

## 📌 Overview
Data analytics and Power BI dashboard for **Alutamarket** — a student-focused 
e-commerce marketplace where students buy and sell across Nigerian universities.

## 📊 Key Metrics (Feb 23 – Mar 22, 2026)
- 🧑‍💻 **1,996** Active Users
- 👀 **26,381** Page Views
- 🆕 **2,006** New Users
- 🌍 Users across **22 countries**
- 📈 **177% user growth spike** in a single week

## 🛠️ Tools Used
- **Firebase** — Data source (user analytics)
- **Power BI** — Dashboard and visualization
- **Excel/CSV** — Data cleaning and preparation

## 📁 Project Structure
- `/data` — Cleaned CSV files from Firebase export
- `/dashboard` — Power BI (.pbix) dashboard file
- `/screenshots` — Dashboard preview images
